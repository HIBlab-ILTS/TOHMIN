Paramerters

Name Explanation Required Type Range Default Unit
Filename Name of the data file to be analyzed. The file name should be specified accurately.
The file extension can be included or omitted. If the specified file name cannot be found, the data will not be loaded.		 yes String - - -
ID A unique identifier for the monitored animals.
This ID will be used to name the output files generated after analysis.
Therefore, duplicate IDs cannot be used within a single analysis session.		 yes String - - -
Group Specifies the experimental or control group designation. yes String - - -
Prehib Start Time Specifies the date and time of the start of the pre-hibernation period.
The analysis will begin from this specified datetime.		 yes Datetime - - -
Hib End Time Specifies the end time of the hibernation period.
If not specified, the program will automatically determine the end of hibernation period based on the analysis criteria.		 no Datetime - - -
Hib Start Tmp Body temparature (Tb) threshold for detecting the first hibernation onset.
This parameter is specifically designed to be more stringent, in other word lower, than the upper_threshold to ensures that only genuine hibernation onset is detected. Without setting this parameter appropriately, false classifications of the hibernation onset could occur by brief Tb drops or accidental hypothermic states during the pre-hibernation period.		 yes Number -10-50 30 °C
Upper Threshold Lower limit threshold of Tb for judging PA state during hibernation period.
Values above this threshold are considered PA state.		 yes Number -10-50 33 °C
Lower Threshold Upper limit threshold of Tb for deep torpor (DT) state.
Values below this threshold are regarded DT state.		 yes Number -10-50 33 °C
Prehib Low Tb Threshold Tb threshold for low Tb event detection during pre-hibernation period, not in hibernation period.
Default value is set to 33ºC.		 yes Number -10-50 33 °C
Hib Start Discrimination Time threshold for determining hibernation onset.
When Tb remains below upper_threshold for longer than this specified duration, the program classifies it as the hibernation onset rather than temporal low Tb event.		 yes Number 120-7200 interval min
Hib End Discrimination Specifes time interval (minutes) for determining the spontaneous end of hibernation( post-hibernation period).
The program identifies the first point of a period where Tb consistently stays above upper_threshold, with temporary drops below upper_threshold allowed if they are shorter than the specified interval.
When these conditions are met, the program marks the first point of this period as the end of hibernation.		 yes Number 20160-43200 interval min
Dead Discrimination Time criterion for mortality determination.
When Tb remains below lower_threshold for the specified duration, the program determines mortality has occurred.
The time of death is set to the first instance when the lowest Tb was recorded during this period.		 yes Number 10080-43200 interval min
PA Discrimination Time criterion threshold for consolidating separated periodic arousal (PA) episodes into a single event, preventing false torpor classifications from minor body temperature fluctuations. yes Number 0-360 interval min
Exclusion Start Time Defines the start time for excluding data from analysis.
Used to remove known artifacts or obvious errors in the data.		 no Datetime - - -
Exclusion End Time Defines the end time for excluding data from analysis.
Used in conjunction with Exclusion End Time to specify the complete period of data to be excluded.		 no Datetime - - -
